Why Dijkstra Beats Greedy

Dijkstra checks all useful possibilities in increasing cost order. It updates the best known distance for every neighbor and only finalizes a node when the shortest cost to it is confirmed.

Greedy routing only picks the next node that looks closest to the destination. That can be quick, but traffic, longer roads, or blockages can make the locally attractive choice worse overall.
